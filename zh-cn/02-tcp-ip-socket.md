## Http Code

- Http Code说明
    - 1XX：信息
    - 2XX：成功
    - 3XX：重定向
    - 4XX：客户端错误
    - 5XX：服务器错误

## 常用端口

|端口  | 协议名|说明|
|------|------|-------|
|端口  | 协议名|文件传输协议（数据）|

|3389 | RDP| 远程桌面|
